# simple_pwa_todo

#### Minimalistic To Do Progressive Web App

Fully functional app here: https://test-pwa-todo.web.app/

## Features
* Local storage usage
* Installable as PWA (Desktop, Mobile)
* Fully adaptive design
* Friendly UX with sounds :)

## Technologies

* React.js, Hooks
* react-bootstrap
* Assembly by React scripts

## Getting started

1. `mvn install`
2. `npm start`
